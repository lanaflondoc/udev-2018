Représentation et simplification d’une fonction booléenne¶

Il est possible de représenter une fonction logique sous la forme d’une table. Cette représentation offre des avantages pour en déduire sa forme normale ou pour réaliser des simplification.

Table de vérité¶

Une table de vérité permet de représenter une fonction logique de la manière suivante :

  • La table compte autant de colonnes que de paramètres à la fonction plus une colonne pour indiquer le résultat de la fonction.
  • La table compte autant de lignes que de combinaisons possibles pour les valeurs des paramètres. Pour, une fonction à n paramètres, la table de vérité associée compte 2 n lignes.

Les tables de Karnaugh¶

Maurice Karnaugh a inventé une méthode basée sur une représentation graphique pour simplifier les équations logiques. Cette méthode fonctionne bien sur un nombre restreint de variables (au plus 5 ou 6).

Une table de Karnaugh est une table à deux entrées. Si la fonction possède 2n variables alors on place en ligne les n premières variables et en colonne les n suivantes. Si la fonction possède un nombre impair de variables, on place les n+1 premières en ligne.

D’une colonne à une autre (et d’une ligne à une autre), on ne peut faire varier la valeur que d’une seule variable.

Pour la fonction à deux variables

on peut construire la table :

f(a, b) a
0 1
b 0 0 0
1 0 1

Pour la fonction à trois variables

on peut construire la table :

f(a, b, c) a b
00 01 11 10
c 0 0 1 1 0
1 0 0 1 0

Note

Pour les valeurs de a et b, notez que pour passer d’une colonne à l’autre, on fait varier soit la valeur de a soit la valeur de b.

Pour la fonction à quatre variables

on peut construire la table :

f(a, b, c, d) a b
00 01 11 10
cd 00 1 0 0 1
01 0 0 0 0
11 0 1 1 1
10 1 0 1 1

Simplification à l’aide d’une table de Karnaugh¶

Pour simplifier une fonction logique à partir d’un tableau de Karnaugh, on recherche les cases adjacentes contenant des 1. On identifie ainsi les blocs les plus grands possibles composés de 1, 2, 4, 8… cases de la table.

La table est refermée sur elle-même. Cela signifie qu’un case sur la dernière colonne est considérée comme adjacente à une case de la première colonne (et réciproquement). De même, une case sur la dernière ligne est considérée comme adjacente à une case de la première ligne (et réciproquement).

Chaque bloc permet de construire un terme de la fonction.

  • Si une variable change de valeur pour ce bloc, elle n’est pas prise en compte. On ne conserve que les variables qui ne varient pas.
  • Pour une variable a qui ne varie pas, si la valeur est 1, on la note et si la valeur est 0, on la note .
  • Chaque variable du terme est multipliée aux autres (relation logique ET)
  • Chaque terme est additionné aux autres (relation logique OU)

f(a, b) a
0 1
b 0 1 1
